多线程
    理解线程和进程
        线程：
            多个程序片段同时去抢占CPU资源 的过程就叫做多线程 多个线程之间内存是共享的
                多线程不能提高程序的执行效率，提高的是CPU的使用率
        进程:
            操作系统上 程序运行的单元 进程之间的数据是互相隔离的 不共享的
            一个进程可以包含多个线程
